Disaster Project

Introduction

The Project is to create web-based platform to show the disaster event on the whole earth. The project can be divided into three parts: Data Scraper, Semantic Analysis and Visualization.

Data Scraper

Data Scraper part is to obtained disaster data from four web sites (I am required to implement the part): 
EMS: https://emergency.copernicus.eu/mapping/list-of-activations-rapid
		
GDACS: MainPage: http://www.gdacs.org/Data: http://www.gdacs.org/datareport/resources/
		
GDELT: MainPage: https://www.gdeltproject.org/Data: http://data.gdeltproject.org/gdeltv2/masterfilelist.txt
		
Relief: https://reliefweb.int/disasters

Semantic Mapping

The part is based on the twitter data. Use semantic analysis to analyze the twitter data. where and what emotions of people to a disaster event.
	This part now is python-based program and can analyze the emotion is positive or negative
