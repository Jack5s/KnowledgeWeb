Use transform Animation with jQuery

It is not allow to use animation() in jQuery to change the value of transform. The solution is to add and remove class of related animation. Also transition(CSS3) must be use in the animation. But you must be careful that all the CSS style related to animation shouldn't repeat in CSS. So I use .div1 and .div2 to show different CSS

HTML

All you need is a div

<div id="div"></div>

CSS

All CSS properties you need to change should be put into a isolated class

#div {
	position: absolute;
	width: 100px;
	height: 100px;
	background-color: blue;
}

.animation {
	transition: all 0.5s linear;
	-moz-transition: all 0.5s linear;
	-webkit-transition: all 0.5s linear;
	-o-transition: all 0.5s linear;
}

.div1 {
	opacity: 0.3;
	transform: scale(1);
}

.div2 {
	opacity: 1;
	transform: scale(0.5);
}

JS

$(document).ready(function() {
// 	set initial style
	$("#div").addClass("div1");
	$("#div").mouseenter(function() {
		$(this).removeClass("div1");
		$(this).addClass("animation");
		$(this).addClass("div2");
	});
	$("#div").mouseleave(function() {
		$(this).removeClass("div2");
		$(this).addClass("div1");
	});
	});
