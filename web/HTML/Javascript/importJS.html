Javascript引用github

html中常需要引用github中raw获取js文件

<script src="https://raw.githubusercontent.com/Jack5s/JavascriptMenu/master/web/JS/MyJS/mymenu.js"></script>

但因为raw.githubusercontent.com在Response中设置了X-Content-Type-Options:nosniff ，告诉浏览器强制检查资源的MIME，进行加载。由于其返回类型text和引用的js类型不同，可能会被浏览器拦截（Cross-Origin Read Blocking，CORB）

使用github链接时，将链接中的raw.githubusercontent.com换成raw.githack.com即可

<script src="https://raw.githack.com/Jack5s/JavascriptMenu/master/web/JS/mymenu.js"></script>
