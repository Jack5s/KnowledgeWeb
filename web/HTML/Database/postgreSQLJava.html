PostgreSQL Connection with Java

Step 1: Import jar package

Download PostgreSQL JDBC driver for java from:
https://jdbc.postgresql.org/
Then you need to change you build path and add the jar package

Step 2: Connect

use following code to connect with your database:

String url = "jdbc:postgresql://hostName/dataBaseName";
String userName = "LiJingwei";
String password = "12345";
Connection connection = DriverManager.getConnection(url, userName, password);
